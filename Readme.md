Revolut test
=======================

1. Сборка
```
npm install
npm run build
```
После сборки виджет можно открыть в браузере запустив файл index.html, либо запустив сервер.

2. Запуск сервера

В dev режиме:
```
npm run dev
```

В prod режиме:
```
npm run start
```

3. Открыть в браузере http://localhost:3000
